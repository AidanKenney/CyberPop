HTML

Every webpage you visit is most and foremost an HTML document.
It is what describes the structure of a webpage to your web browser, so think of it as its skeleton!

CSS

CSS is what makes the webpage beautiful and organised!
It completely changes how your web browser renders the HTML document with shaping, positionning and colors.
